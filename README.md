# SpotTheOdd  (Unity 2D)
## How to Play
# 3x3 grid tile every round .
# One tile is different based on a rule.
# Click the odd tile as fast as possible.
# Clicking the wrong tile reduces your score.
# The game runs continuously until you press the Stop button.
## Rules Implemented
# Color Rule: One tile has a different color while the others tile .
# Rotation Rule: One tile is rotated differently compared to the tile .
## Scoring System
# Scoring is based on reaction time.
# Faster click higher points.
# Slower cick lower points.
